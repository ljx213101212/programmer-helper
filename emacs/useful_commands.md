## Key Bindings  
C - "control"  
M - "meta key (alt)"  


## File  
C-x C-f   Find (open) file.  Emacs asks for file name at bottom of screen.  
C-x C-s   Save file

## Cursor moving

C-a       Move cursor to beginning of line  
C-e       Move cursor to end of line
